# Spam Detection Using Machine Learning

## 📌 Project Overview
This project is a machine learning-based system designed to automatically detect and filter spam messages. By analyzing message content, the model classifies inputs as either **Spam** (unwanted/dangerous) or **Ham** (safe/normal).

## 🎯 Objective
* Create a system that can automatically detect and filter spam messages.
* Clean and prepare text data for analysis.
* Convert text into numerical form using techniques like Bag-of-Words or TF-IDF.
* Train and test a machine learning model to classify messages accurately.

## 🛠️ Tech-Stack
| Technology | Purpose |
| :--- | :--- |
| **Python** | Programming language |
| **Pandas** | Data handling and preprocessing |
| **Scikit-learn** | Machine learning algorithms |
| **Multinomial NB** | Classification algorithm |
| **Matplotlib/Seaborn** | Visualization |
| **Google Colab** | Development environment |

## 📈 Results
The system provides a simple and efficient way to filter unwanted messages:
* **Accuracy**: 99%
* **Performance**: Effective classification as demonstrated by the confusion matrix results.
